12

Dual-mode vertical reframing

TRACK mode

Subject tracking

  • MediaPipe face detection
  • YOLOv8 fallback for full body
  • "Heavy Tripod" stabilization
  • SmoothedCameraman prevents jitter
  • SpeakerTracker handles occlusions
GENERAL mode

Blurred background

  • Preserves full-width content
  • Groups and landscape scenes
  • Gaussian blur behind main frame
  • No subject detection needed
  • Best for multi-person shots

Architecture

Frontend

  • React 18 + Vite 4
  • Tailwind CSS 3.4
  • Encrypted localStorage for API keys
  • Proxy to backend API

Backend

  • Python 3.11 + FastAPI
  • Async job queue with semaphore
  • FFmpeg for all video processing
  • S3 for cloud storage

Getting started

# Clone the repo
git clone https://github.com/your-username/OpenShorts.git
cd OpenShorts

# (Optional) Configure S3 backup
cp .env.example .env

# Launch everything
docker compose up --build

# Open http://localhost:5175

Add your API keys in Settings: Gemini (required), fal.ai (for AI Shorts), ElevenLabs (for dubbing).

19

Configuration

Server-side (.env)

  • AWS_ACCESS_KEY_ID
  • AWS_SECRET_ACCESS_KEY
  • AWS_REGION / AWS_S3_BUCKET
  • MAX_CONCURRENT_JOBS

Client-side (encrypted)

  • GEMINI_API_KEY (required)
  • FAL_KEY (for AI Shorts)
  • ELEVENLABS_API_KEY
  • UPLOAD_POST_API_KEY
Keys encrypted client-side Never stored on server
